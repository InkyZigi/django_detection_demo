Hardware Support

The main hardware technologies adopted by our project include the JetMax robotic arm and NVIDIA Jetson AGX Orin Developer Kit.

*All the information of hardware comes from the retailer or NVIDIA official.

JetMax Robotic Arm

The JetMax robotic arm is a high-performance and flexible mechanical device that can be used in various applications.

It is designed to work seamlessly with the NVIDIA Jetson AGX Orin Developer Kit, providing powerful computing capabilities and advanced control algorithms.

NVIDIA Jetson AGX Orin Developer Kit

The NVIDIA Jetson AGX Orin Developer Kit is a cutting-edge platform that integrates high-performance processing units, graphics processing units, and artificial intelligence accelerators.

It enables developers to build intelligent systems with real-time processing and deep learning capabilities.
